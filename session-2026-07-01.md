## Key Decisions
- Fully client-side — no server, models from CDN, processing in-browser
- `<img>` + `createObjectURL` for mobile compatibility (not createImageBitmap)
- Batch continues past errors/timeouts
- Arabic OCR opt-in only (extra checkbox adds ara+spa)
- Vision only fires on confidence < 30%
- Theme in localStorage('picpulse-theme'), default dark
- cleanLabel splits on ',' to avoid MobileNet synonyms
